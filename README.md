# custom-xcode-templates
Custom XCode file templates to help generate boilerplate code and ensure consistent code patterns

This was developed with the help of the following:

It also used the [VIPER Module Generator by Kaakati on GitHub](https://github.com/Kaakati/VIPER-Module-Generator) for inspiriation.

# Installation

1. Quit Xcode and navigate to `~/Library/Developer/Xcode/` in Finder
2. If a sub-folder titled `Templates` doesn't exist, create it and navigate into it
3. Inside the `Templates` directory, create a new one called `File Templates`
3. Clone this repo directly in the `File Templates` directory
4. Verify installation by opening Xcode, selecting "Create New File," and scrolling down to the bottom to the section titled "Custom"
